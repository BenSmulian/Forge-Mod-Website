Custom Crafting Recipes - Minecraft Modding 1.20.1

Introduction

In this section, we will learn how to add custom crafting recipes for the items and blocks we've created in earlier sections. This allows players to craft your custom items within the game.

Creating a Crafting Recipe

To create a custom crafting recipe, we need to register it using the appropriate registry.

1. Define the Crafting Recipe


    public class ModRecipes {
        public static final DeferredRegister> RECIPE_SERIALIZERS = 
            DeferredRegister.create(ForgeRegistries.RECIPE_SERIALIZERS, Mod.MODID);

        public static final RegistryObject> CUSTOM_RECIPE = RECIPE_SERIALIZERS.register("custom_recipe", 
            () -> new ShapedRecipeSerializer<>(new CustomRecipe()));
    }

2. Register the Recipe in the Game


    public class CustomRecipe implements Recipe {
        // Define crafting pattern here
        @Override
        public boolean matches(Inventory pContainer, Level pLevel) {
            return pContainer.getItem(0).getItem() == ModItems.CUSTOM_ITEM.get();
        }
    }
