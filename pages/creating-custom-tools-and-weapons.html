Minecraft Modding 1.20.1 - Creating Custom Tools and Weapons

Introduction

In this section, we will guide you through creating custom tools and weapons in your Minecraft mod for version 1.20.1. Custom tools can have unique behaviors, such as special abilities or increased durability, while custom weapons can have custom attack power and effects. This is an exciting step in making your mod truly unique!

Creating a Custom Tool

1. Defining the Tool Item

First, let’s create a custom tool. For this example, we will create a custom pickaxe. To do so, we need to define its properties such as its attack damage, durability, and mining speed. You will also need to register your custom tool in the mod’s main class.

Here's the code for defining a custom pickaxe item:

package com.example.mymod.items;

import net.minecraft.item.PickaxeItem;
import net.minecraft.item.ToolMaterial;
import net.minecraft.item.ItemGroup;
import net.minecraft.item.Item;
import net.minecraft.util.registry.Registry;
import net.minecraft.util.Identifier;

public class CustomPickaxeItem extends PickaxeItem {
    
    public CustomPickaxeItem(ToolMaterial material, int attackDamage, float attackSpeed, Settings settings) {
        super(material, attackDamage, attackSpeed, settings);
    }

    // Register the custom pickaxe item
    public static final Item CUSTOM_PICKAXE = new CustomPickaxeItem(CustomToolMaterials.CUSTOM_MATERIAL, 5, -2.8f, new Item.Settings().group(ItemGroup.TOOLS));

    // Register the item
    public static void register() {
        Registry.register(Registry.ITEM, new Identifier("mymod", "custom_pickaxe"), CUSTOM_PICKAXE);
    }
}

In this example:

  • CustomPickaxeItem extends PickaxeItem, allowing us to customize the behavior of the pickaxe, including its attack damage and speed.
  • CustomToolMaterials.CUSTOM_MATERIAL is a custom tool material we define (more on this below).
  • ItemGroup.TOOLS places the pickaxe in the Tools tab of the creative inventory.

2. Defining Custom Tool Material

Next, we need to define a custom tool material for our pickaxe. Tool materials determine the attributes of tools such as mining speed, attack damage, and durability. Let's create our custom material.

package com.example.mymod.items;

import net.minecraft.item.ToolMaterial;
import net.minecraft.item.Items;
import net.minecraft.recipe.Ingredient;

public class CustomToolMaterials implements ToolMaterial {

    public static final ToolMaterial CUSTOM_MATERIAL = new CustomToolMaterials();

    @Override
    public int getDurability() {
        return 2000; // High durability
    }

    @Override
    public float getMiningSpeedMultiplier() {
        return 8.0f; // Fast mining speed
    }

    @Override
    public float getAttackDamage() {
        return 6.0f; // High attack damage
    }

    @Override
    public int getMiningLevel() {
        return 3; // Comparable to diamond
    }

    @Override
    public Ingredient getRepairIngredient() {
        return Ingredient.ofItems(Items.DIAMOND); // Repairs with diamonds
    }
}

In this example, CUSTOM_MATERIAL has high durability, fast mining speed, and a decent attack damage. You can adjust these values as per your mod’s needs.

3. Registering the Tool

We already registered the custom pickaxe in the main class above. To finalize the registration, call the CustomPickaxeItem.register() method inside your mod’s onInitialize() method to ensure it is properly registered when the mod loads.

public class MyMod implements ModInitializer {

    @Override
    public void onInitialize() {
        // Register the custom pickaxe
        CustomPickaxeItem.register();
        System.out.println("Custom Pickaxe Registered!");
    }
}

Creating a Custom Weapon

1. Defining the Weapon Item

Now let’s create a custom weapon. We will create a custom sword with unique attack power and special effects. The code for a basic custom sword is as follows:

package com.example.mymod.items;

import net.minecraft.item.SwordItem;
import net.minecraft.item.ToolMaterial;
import net.minecraft.item.ItemGroup;
import net.minecraft.item.Item;
import net.minecraft.util.registry.Registry;
import net.minecraft.util.Identifier;

public class CustomSwordItem extends SwordItem {

    public CustomSwordItem(ToolMaterial material, int attackDamage, float attackSpeed, Settings settings) {
        super(material, attackDamage, attackSpeed, settings);
    }

    // Register the custom sword item
    public static final Item CUSTOM_SWORD = new CustomSwordItem(CustomToolMaterials.CUSTOM_MATERIAL, 8, -2.4f, new Item.Settings().group(ItemGroup.COMBAT));

    // Register the item
    public static void register() {
        Registry.register(Registry.ITEM, new Identifier("mymod", "custom_sword"), CUSTOM_SWORD);
    }
}

In this example:

  • CustomSwordItem extends SwordItem to create a custom weapon with unique attributes, such as attack damage and speed.
  • CustomToolMaterials.CUSTOM_MATERIAL is used as the material for the sword, just like with the pickaxe.
  • ItemGroup.COMBAT places the sword in the Combat tab of the creative inventory.

2. Registering the Weapon

As with the custom pickaxe, register the sword by calling the CustomSwordItem.register() method in your mod’s onInitialize() method:

public class MyMod implements ModInitializer {

    @Override
    public void onInitialize() {
        // Register the custom sword
        CustomSwordItem.register();
        System.out.println("Custom Sword Registered!");
    }
}
