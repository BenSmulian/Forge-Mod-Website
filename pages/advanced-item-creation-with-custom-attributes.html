Advanced Item Creation - Minecraft Modding 1.20.1

Introduction

In this section, we will explore advanced item creation. You will learn how to create custom items with unique attributes, such as special enchantments or crafting behaviors.

Creating a Custom Item

We'll start by creating a custom item class and registering it.

1. Define the Custom Item Class


    public class CustomItem extends Item {
        public CustomItem(Item.Properties properties) {
            super(properties);
        }
    }

2. Register the Custom Item


    public class ModItems {
        public static final DeferredRegister ITEMS = DeferredRegister.create(ForgeRegistries.ITEMS, Mod.MODID);
        public static final RegistryObject CUSTOM_ITEM = ITEMS.register("custom_item", 
            () -> new CustomItem(new Item.Properties().stacksTo(1).fireResistant()));
    }

Adding Custom Attributes

Now, we'll add a unique attribute to the item, such as an increased attack damage or custom effect.

3. Add a Custom Attribute Modifier


    public class CustomItem extends Item {
        public CustomItem(Item.Properties properties) {
            super(properties);
        }

        @Override
        public void appendHoverText(ItemStack stack, Level world, List tooltip, TooltipFlag flag) {
            super.appendHoverText(stack, world, tooltip, flag);
            tooltip.add(new TextComponent("This item has special powers!"));
        }
    }

Conclusion

In this section, we focused on creating custom items with unique attributes. In the next section, we will look into integrating these items into custom crafting recipes.
