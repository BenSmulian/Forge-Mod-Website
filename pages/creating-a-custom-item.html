Minecraft Modding 1.20.1 - Creating a Custom Item

Introduction

In this section, we will cover how to create a custom item in Minecraft for modding. We will guide you through creating a basic item, adding it to the game, and providing it with custom functionality.

Creating the Custom Item

1. Define the Item Class

Start by creating a new Java class for your item. This will define the behavior of the item when it is used in the game.

package com.example.mymod.items;

import net.minecraft.item.Item;
import net.minecraft.item.ItemGroup;

public class CustomItem extends Item {
    public CustomItem() {
        super(new Item.Settings().group(ItemGroup.MISC)); // Adds item to Misc tab in inventory
    }
}

In this code, we create a new item class called CustomItem. The ItemGroup.MISC places it under the "Misc" category in the inventory. You can modify this to place the item in a different category.

2. Register the Custom Item

Next, you need to register your item so that Minecraft knows about it. This is done in the main mod class. Add the following code to register your custom item.

package com.example.mymod;

import com.example.mymod.items.CustomItem;
import net.minecraft.item.Item;
import net.minecraft.util.Identifier;
import net.minecraft.util.registry.Registry;
import net.fabricmc.api.ModInitializer;

public class MyMod implements ModInitializer {
    public static final Item CUSTOM_ITEM = new CustomItem();

    @Override
    public void onInitialize() {
        Registry.register(Registry.ITEM, new Identifier("mymod", "custom_item"), CUSTOM_ITEM);
    }
}

In this example, we register the custom item under the namespace "mymod" with the identifier "custom_item". You can change "mymod" to the name of your mod and "custom_item" to the name of your item.

3. Add an Item Texture

Now, let's add a texture for the item. First, place your texture image in the src/main/resources/assets/mymod/textures/items/ folder.

The texture file should be named custom_item.png, or whatever you specified when registering the item. Here’s how to add the texture to the game:

src/main/resources/assets/mymod/textures/items/custom_item.png

4. Testing the Custom Item

Now that everything is set up, you can run your mod to test the custom item in Minecraft.

./gradlew runClient

Once Minecraft starts, open the inventory, and you should see your custom item under the "Misc" category. You can also add it to your player’s inventory using commands like:

/give @p mymod:custom_item 1

Next Steps

Now that you’ve created a basic custom item, you can move forward and add additional features like crafting recipes, custom effects, or interaction events. In the next section, we will explore adding advanced functionality to your custom items!
