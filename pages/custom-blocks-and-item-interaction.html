Custom Blocks and Item Interactions - Minecraft Modding 1.20.1

Introduction

In this section, we will create a custom block and make it interact with items. This will include making a block that performs an action when right-clicked or interacted with in specific ways.

Creating a Custom Block

To start, we will define a new block class and register it with Minecraft's block registry.

1. Define the Custom Block


    public class CustomBlock extends Block {
        public CustomBlock(AbstractBlock.Properties properties) {
            super(properties);
        }
    }

2. Register the Custom Block


    public class ModBlocks {
        public static final DeferredRegister BLOCKS = DeferredRegister.create(ForgeRegistries.BLOCKS, Mod.MODID);
        public static final RegistryObject CUSTOM_BLOCK = BLOCKS.register("custom_block", 
            () -> new CustomBlock(AbstractBlock.Properties.of(Material.STONE)));
    }
